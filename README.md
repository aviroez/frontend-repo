# Front End Technical Test from EBUDDY PTE. LTD.

## Initial setup
First, clone from our repository
```bash
git clone https://github.com/aviroez/frontend-repo
```

Second, make sure `NodeJs` is installed and run 
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Backend Setup

You need to make sure that the backend & firebase are ready on [Backend](https://github.com/aviroez/backend-repo)

## Running Apps

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Login / Register

Open [http://localhost:3000/auth](http://localhost:3000/auth) to Login / register
